# Exercise 1: Long Reading

## Purpose
Deep reading comprehension. The user reads a medium-length article and answers 5 questions that test understanding of main ideas, details, and inference.

## Content

### The Article
- 250-350 words (concise — keep it digestible for daily practice)
- Topic: general interest — science, culture, history, nature, society (IELTS Academic style)
- Written or adapted by AI to be at an upper-intermediate level (B2-C1)
- Should feel like a real article, not a textbook excerpt
- Includes a title and source attribution (even if AI-generated, give it a "publication" feel)
- Use short paragraphs (3-4 sentences each) with clear paragraph breaks

### The Questions
- 5 questions total
- Mix of types within the same exercise:
  - **Multiple choice** (4 options, 1 correct): 2-3 questions
  - **True / False / Not Given**: 2-3 questions
- Questions follow the order of the text (first question = first part of article)
- Questions test: main idea, specific detail, inference, author's purpose
- Distractors should be plausible, not obviously wrong

## UI/UX

1. Article displayed at top, full text visible (scrollable if long)
2. Questions below the article, numbered 1-5
3. Multiple choice: radio buttons for each option
4. T/F/NG: three radio buttons (True / False / Not Given)
5. Single "Submit" button at the bottom
6. User can scroll back to the article while answering
7. All 5 must be answered before submit is enabled

## Feedback

After submission:
- Each question shows: user's answer, correct answer, brief explanation (1 sentence)
- Correct answers highlighted green, incorrect red
- Score shown as "X/5"
- No partial credit — each question is right or wrong

## Scoring
- 5 points maximum
- Score stored as integer (0-5)
